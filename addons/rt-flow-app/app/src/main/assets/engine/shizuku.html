⚡ Shizuku · 自我 ADB（ADB 级持久权限）

Android 安全模型禁止 App 凭空给自己授予 ADB/无障碍/全文件等权限。Shizuku 是唯一合规路线——一次性激活后，一个 shell 级特权进程常驻，本 App 即可自我授予一切权限、跑 adb shell 命令，持久稳定（接近有线 USB 的可靠性）。

一键自我授权

用 Shizuku 自动开启：全文件访问、无障碍服务、联系人/短信/通话记录/电话/相册/通知等危险权限——免去逐个手动设置。

  

  

    
ADB Shell（高级）

    
以 shell uid(2000) 执行任意 adb 命令。例：dumpsys battery、wm size、input tap 500 500。

    
    
    

  






  如何一次性激活 Shizuku（任选其一，之后长期有效）：
  
    
    
  1. 无线调试（安卓 11+，无需电脑）：系统设置→开发者选项→开启「无线调试」→ 打开 Shizuku App→「通过无线调试启动」，按提示配对一次。
    2. 
    
  2. 插一次电脑（USB）：电脑装 adb，手机开 USB 调试，运行 Shizuku 提供的 adb shell sh /storage/emulated/0/Android/.../start.sh。
    3. 
    
  3. Root：已 root 直接在 Shizuku 内一键启动。
    4. 
  

  
激活只需一次；重启手机后在 Shizuku 内再点一下启动即可（无线调试方式）。
